# 🚀 YouthPolitics Deployment Guide

## ✅ RECOMMENDED: GitHub Pages (100% FREE Forever)

### Step 1: Create a GitHub Account
1. Go to https://github.com/signup
2. Create a free account
3. Verify your email

### Step 2: Create a New Repository
1. Click the **"+"** icon (top right) → **"New repository"**
2. Repository name: `youthpolitics` (or any name you like)
3. Description: "YouthPolitics - SA Political Education Platform"
4. Make it **Public**
5. ✅ Check **"Add a README file"**
6. Click **"Create repository"**

### Step 3: Upload Your Files
1. Click **"Add file"** → **"Upload files"**
2. Drag and drop ALL your files:
   - index.html
   - sa-politics.html
   - world-politics.html
   - economy.html
   - news.html
   - styles.css
   - script.js
   - data-service.js
   - news-loader.js
   - economy-charts.js
   - manifest.json
3. Write commit message: "Initial website upload"
4. Click **"Commit changes"**

### Step 4: Enable GitHub Pages
1. Go to **Settings** (in your repository)
2. Scroll down to **"Pages"** (left sidebar)
3. Under **"Source"**, select **"main"** branch
4. Click **"Save"**
5. ✅ Wait 2-3 minutes

### Step 5: Get Your Live Link
Your website will be live at:
```
https://YOUR-USERNAME.github.io/youthpolitics/
```

**Example:** If your username is "bongi2026", your link will be:
```
https://bongi2026.github.io/youthpolitics/
```

## 🌟 Alternative Options

### Option 2: Netlify (FREE - Easiest Drag & Drop)

1. Go to https://www.netlify.com/
2. Sign up (free)
3. Click **"Add new site"** → **"Deploy manually"**
4. Drag your **entire Web folder** onto the page
5. ✅ Done! You get a link like: `https://your-site-name.netlify.app`

**Benefits:**
- Instant deployment (30 seconds)
- Auto SSL certificate (https)
- Custom domain support

---

### Option 3: Vercel (FREE - Best Performance)

1. Go to https://vercel.com/
2. Sign up with GitHub
3. Click **"Add New"** → **"Project"**
4. Import your GitHub repository
5. Click **"Deploy"**
6. ✅ Live at: `https://your-site.vercel.app`

## 🔄 How to Update Your Live Site

### If using GitHub Pages:
1. Go to your repository
2. Click on the file you want to edit
3. Click the **pencil icon** (edit)
4. Make changes
5. Click **"Commit changes"**
