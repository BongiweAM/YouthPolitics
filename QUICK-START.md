# ⚡ QUICK START - Get Your Site Live in 5 Minutes

## 🎯 EASIEST METHOD: Netlify Drop

### Steps:
1. **Go to:** https://app.netlify.com/drop
2. **Drag** your entire `Web` folder onto the page
3. **Wait** 30 seconds
4. **✅ DONE!** You get a link like: `https://random-name-123.netlify.app`

**To keep the link permanent:**
- Click "Claim this site"
- Create a free Netlify account
- Rename your site to something like: `youthpolitics.netlify.app`

## 🏆 RECOMMENDED: GitHub Pages (Best for Long-Term)

### Why GitHub Pages?
- ✅ 100% free forever
- ✅ No credit card needed
- ✅ Professional link
- ✅ Easy updates
- ✅ Version history

### Steps:

#### 1️⃣ Create GitHub Account (2 minutes)
```
1. Go to: https://github.com/signup
2. Enter email, password, username
3. Verify email
4. ✅ Done!
```

#### 2️⃣ Create Repository (1 minute)
```
1. Click "+" icon (top right) → "New repository"
2. Name: youthpolitics
3. Make it: Public
4. Check: "Add a README file"
5. Click: "Create repository"
```

#### 3️⃣ Upload Files (2 minutes)
```
1. Click: "Add file" → "Upload files"
2. Drag ALL files from your Web folder:
   ✅ index.html
   ✅ sa-politics.html
   ✅ world-politics.html
   ✅ economy.html
   ✅ news.html
   ✅ styles.css
   ✅ script.js
   ✅ data-service.js
   ✅ news-loader.js
   ✅ economy-charts.js
   ✅ manifest.json
   ✅ README.md (optional)

3. Write message: "Initial upload"
4. Click: "Commit changes"
5. ✅ Files uploaded!
```

#### 4️⃣ Enable GitHub Pages (1 minute)
```
1. Click: "Settings" (in your repository)
2. Click: "Pages" (left sidebar)
3. Under "Source": Select "main" branch
4. Click: "Save"
5. ✅ Wait 2-3 minutes
```

#### 5️⃣ Get Your Link
```
Your site is live at:
https://YOUR-USERNAME.github.io/youthpolitics/

Example: https://bongi2026.github.io/youthpolitics/
```

## 🆘 Troubleshooting

### "My site isn't loading"
- Wait 2-3 minutes after enabling GitHub Pages
- Check that `index.html` is in the root folder (not in a subfolder)
- Make sure repository is Public

### "Images aren't showing"
- Images from Wikipedia should load fine
- If not, check your internet connection

### "Links don't work"
- Make sure all your HTML files are uploaded
- Check that file names match exactly (case-sensitive)
